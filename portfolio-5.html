MUT Chat

Description

  • John F Druhan Team Project 2

  • www.MUT Chat.com

In this project, we will be creating a very mobile friendly web app for the ability to customize playbooks for the online video game Madden19. Our app will pull playbooks from a select amount of teams and give the user access to each teams offensive/defensive plays. Once the user has access to said plays, they wil be able to make their own custom "playlist" from any team.

Prerequesite Technology

  • MySQL

Front End Technology

  • CSS3
  • Javascript

Back End Technology

  • MySQL Database
