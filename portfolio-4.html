Summer Reading List

A Google Book Search App

Description

  • John F Druhan

  • www.BookSearch.com

The client teaches 8th grade English at a local middle school. The client came to me with the desire to create an application that would allow her students to chose which books the class would read for summer break. This App is designed and coded to resemble Google Books Search App using React lifecycle methods to query and display books based on user searches.
